# 📡 Packet Sniffer using Python

## 📌 Description
A Python-based packet sniffer built using Scapy that captures and analyzes real-time network traffic. It detects protocols, DNS requests, HTTP URLs, and possible credential leaks.

---

## 🚀 Features
- Capture live network packets
- Identify TCP and UDP protocols
- Detect DNS requests
- Extract HTTP URLs
- Detect potential credential leaks from packets

---

## 🛠️ Technologies Used
- Python
- Scapy
- Wireshark (for comparison)

---

## ▶️ How to Run

1. Install dependencies: pip install -r requirements.txt


2. Run the sniffer: sudo python sniffer.py


(Use Administrator mode on Windows)
